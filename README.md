# 🎓 Student Analytics Dashboard – Power BI

An interactive Power BI dashboard created to analyze student details, course enrollment, fees collection, and certificate issuance.

## Project Overview

This dashboard provides insights into:

- Total fees collected
- Total student count
- Course-wise fee analysis
- Certificate issued statistics
- Monthly course enrollment trends
- Age group analysis
- Dynamic filtering using slicers

The project helps transform raw student data into meaningful visual insights.

---

## Dashboard Features

✅ KPI Cards for Total Amount and Student Count

✅ Course-wise fee comparison

✅ Certificate issuance gauge chart

✅ Monthly enrollment trend analysis

✅ Age-group distribution

✅ Interactive filters:
- Year
- City
- Course

✅ Mobile responsive layout

✅ Customized themes and formatting

---

## Data Processing Steps

1. Imported student dataset into Power BI

2. Cleaned unwanted/blank values

3. Applied data formatting

4. Created calculated fields and visualizations

5. Added slicers for filtering

6. Customized themes and dashboard layout

7. Adjusted card visuals and formatting

8. Designed mobile view
